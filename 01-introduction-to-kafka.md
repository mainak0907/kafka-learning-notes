# Apache Kafka In-Depth Learning Notes

## Introduction

Apache Kafka is a distributed event streaming platform that is widely used for building real-time data pipelines and streaming applications. It was originally developed by LinkedIn and later open-sourced under the Apache Software Foundation.

### Key Features
- High throughput
- Fault tolerance
- Scalability
- Durability

This repository will contain comprehensive notes on Kafka as we progress through the learning journey.
